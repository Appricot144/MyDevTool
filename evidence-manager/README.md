# 証跡キャプチャツール

単体試験の証跡スクリーンショットを効率的に収集・管理するためのツールです。  
`Alt+PrintScreen` を押すだけで自動保存され、手順番号と連番でファイル名が付きます。

---

## 動作環境

- Windows 10 / 11
- PowerShell 5.1 以上（Windows標準搭載）

---

## 初回セットアップ

PowerShellのスクリプト実行ポリシーを変更します。  
**初回のみ**、PowerShellを管理者として開いて以下を実行してください。

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

---

## 起動方法

`証跡キャプチャ.ps1` を右クリック →「**PowerShellで実行**」

---

## 保存先の変更

スクリプト冒頭の1行を編集してください。

```powershell
$saveFolder = "C:\作業フォルダ\Screenshots"
```

指定フォルダが存在しない場合は自動で作成されます。

---

## ファイル名の形式

```
{手順番号}-{連番}.png
```

| ファイル名 | 意味 |
|---|---|
| `4-1.png` | 手順4の証跡1枚目 |
| `4-2.png` | 手順4の証跡2枚目 |
| `5-1.png` | 手順5の証跡1枚目 |

---

## 基本的な使い方

1. ツールを起動する
2. スピナーで手順番号をセットする
3. 試験対象の画面で `Alt+PrintScreen` を押す → **自動で保存＆サムネイル表示**
4. 手順が進んだら `Ctrl+↑` で手順番号を増やす
5. 以降くり返す

---

## ショートカットキー

### メインウィンドウ

| キー | 動作 |
|---|---|
| `Alt+PrintScreen` | アクティブウィンドウをキャプチャして自動保存 |
| `Ctrl+↑` | 手順番号 +1（連番は1にリセット）|
| `Ctrl+↓` | 手順番号 -1（連番は1にリセット）|
| `Enter` | 選択中のサムネイルを拡大表示 |

### 拡大表示ウィンドウ

| キー / ボタン | 動作 |
|---|---|
| `←` / `◀ 前へ` | 前の画像へ |
| `→` / `次へ ▶` | 次の画像へ |
| `Esc` | 閉じる |

---

## 機能詳細

### 自動保存
クリップボードのシーケンス番号を500ms間隔で監視しています。  
`Alt+PrintScreen` を押すたびにシーケンス番号が更新されるため、**押した回数 = 保存枚数**になります。  
テキストなど画像以外のコピー操作は無視されます。

### 監視の一時停止
`⏸ 監視停止` ボタンで監視を止められます。  
試験と無関係な操作をする際に使用してください。`▶ 監視再開` で再開できます。

### サムネイル表示
保存するたびにウィンドウ内にサムネイルが追加されます。  
クリックまたは `Enter` で拡大表示できます。

### 拡大表示の順送り
拡大表示ウィンドウ内で `←` `→` キーまたはボタンで画像を切り替えられます。  
下部バーに現在のファイル名と通し番号（例: `4-2.png  (3 / 7)`）が表示されます。
